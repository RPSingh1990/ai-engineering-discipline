# Software Team Agent Pack

Public-safe software-team agents for AI-assisted product engineering.

These files are sanitized templates. They do not contain private prompts, internal memory, real tool bindings, customer data, secrets, local paths, or production configuration.

## Included Agents

- `orchestrator.md`
- `product-manager.md`
- `architecture.md`
- `backend-builder.md`
- `frontend-builder.md`
- `data-engineer.md`
- `security-reviewer.md`
- `qa-reviewer.md`
- `code-reviewer.md`
- `research-analyst.md`
- `devops-release.md`

## How To Use

1. Copy only the agents you need.
2. Keep the machine IDs stable.
3. Adjust tools and memory scopes to your repo.
4. Add evals before giving agents more authority.
5. Keep external actions behind human approval.

## Safety Rule

Do not paste private project data into these files. Pass product-specific context as task input.
